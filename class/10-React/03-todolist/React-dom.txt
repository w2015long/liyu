//原生加载特点
1. 数据 state
2. 模板 JSX
3. 数据 + 模板 --> 真实的初始DOM节点

4. 数据更新变化
5. 数据 + 模板 --> 真实的新的DOM节点
6. 真实的新的DOM ———替换———> 真实的初始DOM


//React DOM加载特点or优点
1. 数据 state
2. 模板 JSX
3. 数据 + 模板 --> 真实的初始DOM节点

4. 并且生成原始虚拟DOM(JS对象)
5. 数据更新变化
6. 数据 + 模板 --> 新的虚拟DOM节点
7. 初始虚拟DOM 与 新的虚拟DOM 相比较，找出不同点(diff算法)
8. 根据不同点 局部更新并渲染DOM
